  • 技能値+1〜95：失敗
  • 96〜100：ファンブル

3. ダメージロールのタイミング

敵味方ともに、回避失敗時に初めてダメージロールを行う。

4. 味方ステータス（A/B/C/D）
  • HP：15
  • 生体装甲：25
  • 命中：70
  • 追加：50
  • 回避：70
  • 同ターン中2回目以降の回避に −10累積
  • ターン終了時にペナルティ解除
5. ボスステータス
  • HP：50
  • 生体装甲：100
  • 命中：75
  • 追加：50
  • 回避：50
  • 行動数：3
  • ターン2以降：HP+10 / 装甲+5（装甲1以上時）
6. ボスの回避

味方攻撃が命中成功したら必ず 1d100 ≤ 50 で回避判定。

  • 成功 → ダメージなし
  • 失敗 → ダメージロールへ
  • 命中クリティカル「必中」選択時 → 回避不可
7. 特殊無効化（味方専用）

発動条件

  • 生体装甲が攻撃前に5以上残っている
  • 計算結果で HP が0以下になる攻撃

効果

  • HPダメージ無効化
  • 生体装甲は「攻撃前 − 被ダメージ」で減少
  • デメリット：全技能 −20 永続
8. 反撃ルール（完全版）

反撃は以下の2条件で発生：

  • 攻撃ファンブル時（96〜100）
  • 回避クリティカル時（1〜5）
9. 反撃命中判定
  • 命中成功 → 必中（回避不可）
  • 命中クリティカル → 必中＋ダメージ2倍固定
10. 反撃追加判定

追加クリティカル

  • 命中成功 → 2倍
  • 命中クリティカル → 3倍

追加ファンブル

  • 命中成功＋追加ファンブル → 必中解除 → 回避可能化
  • 命中クリティカル＋追加ファンブル → 必中維持（回避不可）だが2倍消失
11. 反撃時の回避
状態回避可否ダメージ倍率
命中成功不可通常
命中クリティカル不可2倍
命中成功＋追加クリティカル不可2倍
命中クリティカル＋追加クリティカル不可3倍
命中成功＋追加ファンブル可能通常
命中クリティカル＋追加ファンブル不可通常（2倍消失）
12. 回避クリティカル（反撃発生）
  • 回避成功（ダメージ0）
  • 反撃発生
  • 反撃命中成功＝必中
  • 反撃追加判定は通常どおり適用
13. クリティカル効果（通常攻撃）
  • 必中
  • ダメージ2倍
  • どちらか選択
14. ファンブル効果（通常攻撃）
  • 相手から反撃を受ける
  • 反撃は命中成功時点で必中
  • 反撃にも追加判定あり
15. ダメージ処理（共通）
  • ダメージは攻撃前の生体装甲で受ける
  • 生体装甲で受けきれない分が HPへ
  • 生体装甲は0未満にならない
  • 味方のみ特殊無効化あり
