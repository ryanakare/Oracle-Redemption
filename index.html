🔮 Oracle Redemption ― 背景設定（世界観用）

✦ 起源：堕ちた神託者

かつて世界には、未来を読み解く者たちがいた。
彼らは 「オラクル」 と呼ばれ、神々の声を聞き、災厄を未然に防ぐ存在だった。

しかし、その中の一人が “誤った神託” を告げてしまう。

その神託は国を滅ぼし、数万の命が失われた。

彼は 「神託者の罪」 として追放され、自らも声を失い、未来を視る力を封じられた。

✦ 贖罪の始まり

数十年後、世界に再び災厄が迫る。
神々は沈黙し、オラクルはすでに存在しない。

そのとき、かつての“堕ちた神託者”の魂に、再び微かな声が届く。

「汝の罪を贖い、世界を救え」

彼は再び立ち上がる。
これは、神々が与えた最後の機会――
Oracle Redemption（神託の贖罪）。

✦ 能力の本質

Oracle Redemptionを持つ者は、完全な予知能力ではなく、
“未来の断片” を読み取る。

しかしその力は、過去の罪を背負う者にしか宿らない。

  • 未来の危機を察知する
  • 味方の行動を最適化する
  • 敵の攻撃を予兆として感じ取る
  • 自身のMP消費が軽減される（魔の泉と相性抜群）

力の源は神ではなく、「贖罪の意志」そのもの。

✦ 世界観への影響

Oracle Redemptionを持つ者は、人々から恐れられ、同時に求められる。

  • 過去の罪を知る者は彼を忌避する
  • 新たな災厄を恐れる者は彼を頼る
  • 神官たちは彼を“神の試練”と呼ぶ
  • 魔術師たちは彼の力を研究対象とする

彼の存在は、世界に “希望” と “恐怖” を同時にもたらす。

✦ キャラ背景としての使い方

  • 過去に重大な失敗をした
  • その罪を背負い続けている
  • しかし再び立ち上がる強さを持つ
  • 神々ではなく“自分自身”の意志で未来を変える
